# 考前衝刺版總整理（六）：XRD 繞射光譜

---

# 一、總架構

> XRD 是利用 X-ray 與晶體作用產生繞射來分析晶體結構。當滿足 Bragg's law，且 path difference = nλ 時，會產生建設性干涉形成繞射峰。每個 peak 對應一組晶面 (hkl)。XRD 圖譜可作為物質 fingerprint，用來做物相分析、結晶度判斷、晶格分析、晶粒大小估算等。

---

# 二、必背公式

## Bragg's Law

**nλ = 2d sinθ**

- n：繞射階數
- λ：X 光波長
- d：晶面間距
- θ：Bragg angle

> **當 path difference = nλ 時，產生建設性干涉，所以出現 peak。**

---

# 三、最容易考簡答的觀念

## XRD 在幹嘛

- phase identification
- crystallinity
- lattice parameters
- crystallite size
- residual stress

> **XRD pattern 是材料的 fingerprint。**

---

## X-ray 怎麼來

**加熱 cathode 放出電子 → 高電壓加速 → 撞擊 anode → 產生 characteristic X-ray（Kα）**

> **XRD 使用 characteristic X-ray，常見為 Kα。**

---

## 為什麼不同物質有不同 pattern

**因為不同晶體的晶面間距 d 不同，所以滿足 Bragg's law 的角度不同，產生不同繞射圖譜。**

---

# 四、容易搞混但必考的觀念

## 粉末 XRD 為什麼會形成圓環

**粉末樣品中晶粒方向隨機分布，因此不同方向都可能滿足 Bragg 條件，形成繞射環。**

## Peak 代表什麼

**每個 peak 對應一組晶面（hkl）。**

## Miller Index

**Miller index (hkl) = 晶面與座標軸截距的倒數，化成最簡整數。**

## 晶體 vs 非晶

- crystalline → 尖銳 peaks
- amorphous → 寬廣 hump

---

# 五、進階重點

## Synchrotron XRD 為什麼強

**Synchrotron XRD 具有高光子通量、高解析度、快速量測，並可提供高角度資訊。**

**高角度資訊 = 高 2θ 區域的訊號，對應較小 d，可提供更細的結構資訊。**

記憶法：
- 29 頁：快不快（synchrotron 用更短時間量到圖）
- 30 頁：準不準（synchrotron 結果更接近 theoretical/database）

---

## Rietveld Refinement

**Rietveld refinement 是用 least squares 擬合理論與實驗的整個 XRD 圖譜，不是只看單一 peak。**

---

## Scherrer Equation

**Scherrer equation 用 peak 寬度估算 crystallite size。**

> **Peak 越寬，晶粒越小。**

白話版：**小晶粒 → 可連續干涉的範圍變短 → 干涉不集中 → peak 變寬。**

---

# 六、考前最後速背版

### 核心
- XRD 用來分析晶體結構
- 滿足 Bragg's law 時產生繞射 peak
- XRD pattern 是 material fingerprint

### Bragg's Law
- nλ = 2d sinθ
- path difference = nλ → constructive interference → peak

### XRD 用途
- phase identification、crystallinity、lattice parameters、crystallite size、residual stress

### X-ray 來源
- cathode 放電子 → 高電壓加速 → 撞 anode → characteristic X-ray（Kα）

### 粉末 XRD
- 晶粒方向隨機 → 形成繞射環

### Peak 與 Miller Index
- 每個 peak 對應一組 (hkl)
- Miller index = 截距倒數，化整數

### 晶體 vs 非晶
- crystalline：sharp peaks
- amorphous：broad hump

### Synchrotron XRD
- 強、快、解析度高、可得高角度資訊

### Rietveld
- least squares、fit entire pattern

### Scherrer
- 用 peak 寬度算晶粒大小
- peak 越寬 → 晶粒越小
