### 📐 從 Examples 看老師期望的答題格式

**寫技術題時應該包含**：
1. **原理機制**：從物理/化學層面解釋「發生什麼事」
2. **明確比較框架**：兩個技術放對照，逐點列出差別
3. **關鍵物理量**：不要只寫名詞，要寫**為什麼**（如：全反射 → evanescent wave → 穿透 ~1 μm）
4. **適用情境**：什麼樣品用什麼技術，為什麼
5. **具體範例**：舉一個實際物件（螢光筆 / EXIT 指示牌 / 粉末催化劑）

**避免的寫法**：
- ❌ 只寫名詞解釋（「螢光是光發射」）
- ❌ 只背定義不說原因（「磷光壽命長」沒解釋為什麼）
- ❌ 把兩個東西分開寫（沒有放在一起比較）

### 📖 兩個示範答案的書寫結構分析

**ATR vs DRIFT 示範**的結構：
```
1. 先點出兩者都是 IR 取樣技術 → 定位
2. 主要差別（一句話）：IR 與樣品交互方式 + 樣品型態
3. ATR：物理機制（全反射）+ 晶體材料 + 適用樣品 + 樣品製備
4. DRIFT：物理機制（散射+漫反射）+ 適用樣品 + 樣品製備 + 深度對比
```

**Fluorescence vs Phosphorescence 示範**的結構：
```
1. 點出兩者共同本質：都是 photoluminescence
2. 主要差別（一句話）：「螢光即時、磷光延遲」
3. 螢光：能階躍遷（singlet→singlet）+ 速度快 + 範例（螢光筆）
4. 磷光：能階躍遷（ISC + triplet→singlet）+ 為何慢（forbidden transition）+ 範例（EXIT 指示牌）
```

**共同結構**：
```
[定位同類] → [一句話點出差別] → [物理機制] → [適用/應用] → [具體範例]
```
**這個結構可以套用到任何「比較兩個技術」的題目**。

### 🎯 實戰演練 — 其他可能考題的答題框架

考試可能考的其他比較題（用同樣的結構回答）：

**「比較 FTIR 與 Dispersive IR」**
```
→ 都是 IR 吸收光譜 → 差別在波長選擇方式
→ Dispersive：grating monochromator 一次一個波長（慢）
→ FTIR：Michelson interferometer 同時全頻段 + FFT（快、SNR 好）
→ FTIR 三大優勢：Fellgett / Jacquinot / Connes
→ 範例：現代分析都用 FTIR
```

**「比較 XANES 與 EXAFS」**
```
→ 都是 XAS（X-ray absorption spectroscopy）→ 差別在能量區段
→ XANES：edge ±10-40 eV，多重散射主導，給電子結構
→ EXAFS：edge +40 eV 以上，單散射近似，給幾何結構
→ XANES 資訊：氧化態、d 軌道填充、對稱性
→ EXAFS 資訊：鍵長、配位數、鄰近原子
→ 範例：Mn K-edge 氧化態判斷（XANES）、EXAFS 求 Cu-O 鍵長
```

**「比較 DTGS 與 MCT detector」**
```
→ 都是 FTIR 偵測器 → 差別在偵測原理與適用情境
→ DTGS：熱型（pyroelectric），室溫操作，線性響應寬，高通量好
→ MCT：量子型（HgCdTe），液氮冷卻，低通量高靈敏，動力學測量好
→ 範例：靜態定量用 DTGS，快反應動力學用 MCT
```

**「比較 Prism 與 Grating monochromator」**
```
→ 都是波長選擇器 → 差別在色散物理機制
→ Prism：折射率隨波長變化，色散非線性
→ Grating：繞射分光，色散線性
→ 現代儀器主流是 grating（解析度好、線性）
```

## 🎯 過往考點 A：ATR vs DRIFT（IR 取樣技術差別）

**必考回答框架**：

| 項目             | **ATR**（Attenuated Total Reflectance） | **DRIFT**（Diffuse Reflectance IR FT） |
| -------------- | ------------------------------------- | ------------------------------------ |
| **IR 與樣品作用方式** | **全反射（Total Reflection）**在高折射率晶體中     | **散射 + 漫反射**於粗糙表面                    |
| **關鍵物理**       | 產生 **evanescent wave** 穿透樣品 ~1 μm     | IR 穿入粉末顆粒內部多方向散射                     |
| **常用晶體/材料**    | 鑽石（Diamond）、鍺（Ge）、碲化鋅（ZnSe）、Silicon   | KBr（稀釋劑，非必用）                         |
| **適用樣品**       | 流體、膏狀、柔軟固體、聚合物薄膜、液體、生物樣品              | 粉末、催化劑、粗糙表面固體、錠片                     |
| **樣品製備**       | **幾乎不需製備**，直接放置即可                     | 需**磨細**，常以 KBr 稀釋以提升訊號               |
| **量測深度**       | 淺（μm 級，僅表面資訊）                         | **較深，可反映 bulk（塊體）特性**                |
| **訊號機制**       | 晶體與樣品緊密接觸，光在晶體中全反射時逸波被樣品吸收衰減          | 偵測樣品散射回來的 diffuse reflected light    |

**一句話記憶**：
- **ATR = 表面 + 全反射 + evanescent wave + 免壓片**
- **DRIFT = 內部 + 漫反射 + 粉末 + bulk 資訊**

**關鍵公式（ATR 穿透深度）**：
```
d_p = λ / (2π n_crystal √(sin²θ − (n_sample/n_crystal)²))
θ_c = sin⁻¹(n_sample/n_crystal)  (臨界角)
```

### 🎯 考點 B：Fluorescence vs Phosphorescence

**核心定義**：兩者都是**光致發光（Photoluminescence, PL）**，差別在**電子自旋狀態**與**發光持續時間**。一句話總結：**螢光是即時的，磷光是延遲的**。

**必考比較表**：

| 項目 | **Fluorescence（螢光）** | **Phosphorescence（磷光）** |
|---|---|---|
| **能階躍遷** | Singlet → Singlet（S₁ → S₀） | Triplet → Singlet（T₁ → S₀） |
| **自旋是否改變** | 不改變（spin-allowed） | **需 spin flip**（spin-forbidden） |
| **路徑** | 吸收後直接在 singlet 態，快速返回 | 吸收 → S₁ → **ISC（Intersystem Crossing）** → T₁ → S₀ |
| **發生時間** | ~ns（10⁻⁹ 秒，10¹⁰–10⁷ s⁻¹） | ~ms 到小時（10⁵–10⁻³ s⁻¹） |
| **移除光源後** | 立刻停止 | **持續發光**（"glow-in-the-dark"） |
| **發射能量** | 較高、波長較短 | 較低、波長較長（紅移） |
| **範例** | **螢光筆**、GFP、fluorescein | **緊急出口指示牌**（EXIT）、夜光材料 |

**關鍵機制：ISC（Intersystem Crossing）**
- 從 singlet 激發態 S₁ **翻轉自旋** 到 triplet 激發態 T₁
- 這個過程本身是 **「forbidden transition」**（需自旋翻轉）
- 但因為自旋-軌道耦合（spin-orbit coupling）或重原子效應，仍可發生
- **T₁ → S₀ 也是 forbidden transition** → 所以磷光很慢、壽命很長

**為什麼螢光比磷光常見？**
- k_IC > k_ISC（Internal Conversion 比 Intersystem Crossing 快）
- ISC 被禁止 → 多數分子難到達 T₁ → 螢光是主要衰減路徑
- **含重原子**（Br、I、Pt、Pd、Ir 錯合物）時，spin-orbit coupling 增強 → k_ISC 提高 → 磷光顯著

**典型應用**：
- Fluorescence：螢光筆、GFP 生物標記、螢光染料、量子點
- Phosphorescence：緊急出口指示牌、夜光錶、OLED（重原子磷光分子）

## 1. 光的基本物理（L1）

### 電磁波譜

γ → X → UV → **Vis (380–780 nm)** → IR → Microwave → Radio

- UV：200–400 nm；Vacuum UV：10–180 nm
- Visible：400–780 nm
- Near IR：0.78–2.5 μm；Mid IR：2.5–50 μm；Far IR：50–1000 μm
- **Mid IR 最常用**：2.5–15 μm (4000–670 cm⁻¹)

### 關鍵公式

```
c = λν                 (光速 = 波長 × 頻率；頻率不變，波長隨介質改變)
v̄ = 1/λ  (cm⁻¹)        (波數)
E = hν = hc/λ = hc·v̄   (光子能量)
h = 6.63 × 10⁻³⁴ J·s
1 eV = 1.60 × 10⁻¹⁹ J
```

### 線性 vs 非線性光學

- 線性：P = ε₀·χ⁽¹⁾·E（reflection、refraction、absorption、scattering）
- 非線性：P = ε₀(χ⁽¹⁾E + χ⁽²⁾E² + χ⁽³⁾E³ + …)（SHG、SFG，需高強度）

### 光與物質交互作用的三種過程

**Absorption、Emission、Scattering**

### 量子變化對應的光譜區

| 量子變化 | 光譜類型           |
| -------- | ------------------ |
| 自旋     | NMR / ESR          |
| 配向     | Microwave          |
| 組態     | IR                 |
| 電子分布 | **UV-Vis / X-ray** |
| 核組態   | γ-ray              |

## 2. Beer-Lambert Law（考必背）

```
A = log(P₀/P) = −log T = ε·b·c
```

- 適用**稀溶液**（< 0.01 M），limiting law
- 三大偏差：real（高濃度）、chemical、instrumental（stray light + polychromatic）
- Stray light：A' = log[(P₀+P_s)/(P+P_s)] → 高 A 時低估
- A > 2.0 不可靠

## 3. 波長選擇器 & 偵測器（L1）

### Filter vs Monochromator

|        | Filter               | Monochromator    |
| ------ | -------------------- | ---------------- |
| 靈活性 | 固定（換硬體）       | 電子調諧可連續掃 |
| 靈敏度 | **較高**（光損失少） | 較低             |
| 成本   | 便宜                 | 貴               |

### Filter 類型

- Absorption filter（染料，便宜寬頻）
- Interference filter（thin film，1–10 nm，對角度敏感）
- Bandpass / Longpass / Shortpass / Notch

### Monochromator

- Prism（色散非線性）
- Grating（色散線性、現代主流，高密度 → 高解析度）

### SNR 改善

- Ensemble averaging：(S/N)\_n = √n × (S/N)\_i（資料點平方根律）
- 四種雜訊：thermal (Johnson)、shot、flicker (1/f)、environmental
- 降 thermal noise：降電阻、降溫（例 298 K → 液氮 77 K）

## 4. IR 光譜（L1 + L2 + Supp-IR）

### 核心原理

- IR 頻率符合共價鍵的自然振動頻率
- **Selection rule**：分子振動時 **dipole moment 必須改變**
- Homonuclear（H₂/O₂/N₂）→ 永遠 IR inactive
- Heteronuclear（HCl/CO）→ IR active

### Hooke's Law

```
v̄ = (1/2πc)·√(f/μ)
μ = M₁M₂/(M₁+M₂)
```

- f 大（鍵強）→ 波數高：C≡C 2150 > C=C 1650 > C–C 1000
- μ 小（原子輕）→ 波數高：C–H ~3000 > C–Cl ~700
- **aromatic C–H > aromatic C–C** 原因：μ_CH ≪ μ_CC

### 振動自由度

- 非線型：**3N − 6**
- 線型：**3N − 5**

### CO₂ 典型考題

- 線型 3×3−5 = **4 個振動模式**
- 對稱伸縮（IR inactive）+ 反對稱伸縮（IR active, 2350 cm⁻¹）+ 兩個 degenerate 彎曲（IR active, 667 cm⁻¹）
- **IR 光譜只有 2 條吸收帶**

### H₂O 典型考題

- 非線型 3×3−6 = **3 個振動模式**
- 所有 3 個都是 IR active（O-H 對稱伸縮、反對稱伸縮、剪式彎曲）
- 角度 104.5°，有永久淨偶極

### IR 區段特徵帶（考前必記）

| 範圍 cm⁻¹ | 特徵                        |
| --------- | --------------------------- |
| 3500–3200 | O–H（醇寬峰）               |
| 3300–2500 | O–H（羧酸，超寬）           |
| 3500–3300 | N–H                         |
| 3100–3000 | C–H (sp², aromatic, alkene) |
| 3000–2850 | C–H (sp³, alkane)           |
| 2260–2100 | C≡C、C≡N                    |
| 1820–1650 | **C=O**（強峰）             |
| 1680–1620 | C=C                         |
| 1600–1400 | aromatic C=C (ring)         |
| 1400–600  | **Fingerprint region**      |

### Aldehyde 特徵

C–H stretch 2830–2695 cm⁻¹（**2720 shoulder**）

## 5. FTIR（L2 必考）

### 與 grating IR 的差別（**Q2 exercise 題型**）

|        | Grating IR         | **FTIR**                     |
| ------ | ------------------ | ---------------------------- |
| 光學   | Monochromator      | **Michelson interferometer** |
| 量測   | 一次一個波長（慢） | **同時全頻段**               |
| 速度   | 分鐘               | 秒                           |
| SNR    | 較差               | **較好**                     |
| 解析度 | 較差               | 較好（Δv̄ = 1/δ_max）         |

### FTIR 三大優勢

1. **Fellgett (multiplex)**：全頻段同時測
2. **Jacquinot (throughput)**：無 slit 光量大
3. **Connes**：He-Ne 校正波數精準

### Michelson 原理

Source → Beam splitter → {Fixed M₁, Moving M₂} → recombine → sample → detector

- OPD δ = 2Δ
- I(δ) = ∫ B(v̄) cos(2πδv̄) dv̄
- **FFT** → B(v̄)
- 解析度 Δv̄ = 1/δ_max

### IR 光源與偵測器

**Sources**：Globar (SiC)、Nernst glower、tungsten-halogen、高壓 Hg (far-IR)

**Detectors**：

- **DTGS**（pyroelectric, 熱型, 室溫）：**靜態、定量、高通量線性**
- **MCT (HgCdTe)**（quantum, 77 K）：**動力學、低通量、高靈敏度、容易飽和**
- **MCT 在高通量會飽和** → 用 neutral density filter 降光

### Detectivity D\*

- cm·Hz^(1/2)/W，越高越靈敏
- MCT-A ≈ 6.4×10¹⁰ >> DLaTGS ≈ 2.7×10⁸（**差 200 倍**）

### Sampling 技術

- **Transmission**：KBr pellet（1–2 mg + 100–150 mg KBr）、Nujol mull、液體 cell
- **ATR**：利用全反射產生 evanescent wave，穿透 ~1 μm；晶體用 Ge/Si/ZnSe/Diamond（都是高折射率 >2.4）
- **DRIFT**：粉末漫反射，無需壓片
- Critical angle：θ_c = sin⁻¹(n_s/n_c)
- d_p = λ / (2π n_c √(sin²θ − (n_s/n_c)²))

## 6. UV-Vis 光譜（L3）

### 原理

- 激發 valence electrons 到激發態
- 三種主要躍遷：
  - **π → π\***：不飽和化合物
  - **n → π\***：有 lone pair
  - **d–d**：過渡金屬錯合物顏色來源

### Spin selection rule：**ΔS = 0**

- High-spin d⁵：5 電子平行 → 任何 d-d 都改變 spin → **spin-forbidden** → 顏色淡

### 為什麼吸收峰是寬帶

電子能階疊加振動與旋轉能階 → 許多躍遷的包絡線

### Beer-Lambert Law 定量

A = εcl → 建校正曲線 → 未知樣濃度

### UV-Vis 儀器組件

**三大光源**：

- **D₂ lamp**：185–400 nm UV，壽命 1000 hr
- **Tungsten-halogen**：350–3000 nm Vis-NIR，壽命 10000 hr
- **Xenon flash**：185–2500 nm 單一光源全段

**Cuvette 材質**（重要！）：

- **Quartz**：170–2700 nm，**UV 必用**
- Glass：334–2500 nm，**不能 UV**
- Plastic：340–800 nm，便宜拋棄式

**溶劑**：UV 首選水（透過 < 190 nm）

**Dispersion**：prism（色散非線性，老派） / **grating（現代主流）**

**Dispersive power**：Δn 越大越能分離；diamond 最強

**偵測器**：

- **PMT**：200–900 nm，高靈敏度、稀樣品最佳
- **Silicon photodiode**：170–1100 nm，寬動態、便宜
- **Diode array / CCD**：同時量全譜
- **InGaAs**：800–2500 nm (NIR)
- **PbS**：1000–3500 nm (NIR)

### Single vs Double beam

- Single：簡單，需分次量 reference 與 sample
- Double：同時量兩路，自動扣漂移

### UV-Vis 優缺點

**優點**：廣、定量好、快、非破壞、水相 OK、可測固體（diffuse reflectance）
**缺點**：結構資訊差、峰重疊、溶劑干擾、UV 溶劑透明度有限、不是所有化合物都吸收

### Tauc plot（band gap）

```
(α·hν)^(1/γ) vs hν
```

- γ = 1/2（direct allowed）、2（indirect allowed）、2/3、3
- Amorphous：(α·hν)^(1/2) = B(hν − Eg)
- 線性外插到 x 軸截距 = Eg

### Fluorescence vs UV-Vis absorption 哪個對稀樣靈敏？

**Fluorescence** 靈敏 10³–10⁶ 倍。UV-Vis 稀樣品 I ≈ I₀ 時 log(I₀/I) → 0 儀器雜訊掩蓋；fluorescence 在近零背景測絕對發光。

## 7. X-ray Absorption Spectroscopy (L4)

### 核心原理

X 光能量 ≥ core electron binding energy → 光子被吸收、電子被踢出 → **absorption edge**

### Absorption edges

| Edge | Shell |
| ---- | ----- |
| K    | 1s    |
| L₁   | 2s    |
| L₂   | 2p₁/₂ |
| L₃   | 2p₃/₂ |

spin-orbit coupling 讓 2p 分裂成 L₂、L₃。

### Selection rules

- **Dipole Δl = ±1**
- K edge: 1s → p
- L edge: 2s → p; 2p → s, d
- **K edge 1s→3d 是 dipole-forbidden**（只能 quadrupole pre-edge）
- **L₂,₃ edge 2p→3d 是 dipole-allowed** → 對 3d 過渡金屬更敏感
- Quadrupole Δl = ±2 much weaker

### 為什麼不同元素有不同 edge energy（**Q2 考題**）

每元素 core electron 的 binding energy 由核電荷決定，各不相同 → XAS 具元素專一性

### XANES vs EXAFS（**Q2 考題**）

|      | **XANES**                      | **EXAFS**                      |
| ---- | ------------------------------ | ------------------------------ |
| 範圍 | edge +10~40 eV                 | edge +40 eV 以上               |
| 資訊 | **氧化態、d 軌道填充、對稱性** | **鍵長、配位數、鄰近原子種類** |
| 散射 | 多重散射                       | 單散射近似                     |

### 能量分類

- **Hard > 5 keV**：過渡金屬 K / 重元素 L；大氣下可 in situ
- **Tender 2–5 keV**：P, S K-edge；需 He-purged
- **Soft < 2 keV**：輕元素 K、3d 過渡金屬 L；需 UHV

### Synchrotron 必要性

- 比 X-ray tube 強 **10⁶ 倍**
- 連續寬頻、可調能量、高亮度
- **NSRRC TPS：3 GeV，2016 啟用，周長 518 m**

### 量測

- μ·x = ln(I₀/I_t)
- Monochromator：**Si single crystal + Bragg**
- Detector：Gas ionization chamber、Si SSD、Lytle

## 8. Emission / Luminescence / Fluorescence（L4）

### Kirchhoff's Law

「emissivity = absorptivity」：好吸收體也是好發射體

### Luminescence 分類

Photo、Cathodo、Chemi、Electro、Mechano、Radio、Thermoluminescence

### XES（X-ray Emission Spectroscopy）

"Photon-in, photon-out"：探**occupied** 電子態

- Core hole creation → electronic decay → photon emission
- Kα (2p→1s)、Kβ (3p→1s)、Kβ₂,₅ (valence→1s)
- 與 XAS（探 unoccupied）互補

### Fluorescence vs Phosphorescence（**Q3 考題**）

|            | **Fluorescence**      | **Phosphorescence**             |
| ---------- | --------------------- | ------------------------------- |
| Spin state | **S₁ → S₀** (singlet) | **T₁ → S₀** (triplet → singlet) |
| 自旋變化   | 不變 (allowed)        | 需翻轉 (forbidden)              |
| Lifetime   | **ns (10¹⁰–10⁷ s⁻¹)** | **ms–h (10⁵–10⁻³ s⁻¹)**         |
| 移光源後   | 立刻停                | **仍發光（glow-in-dark）**      |
| 發射能量   | 高、波長短            | 低、波長長（紅移）              |
| 範例       | 螢光筆、GFP           | EXIT 指示牌                     |

### ISC (Intersystem Crossing)

S₁ → T₁，禁止但靠 spin-orbit coupling 或重原子效應發生

### Singlet vs Triplet

- Singlet S=0, multiplicity 1, 電子配對
- Triplet S=1, multiplicity 3, 電子平行

### Stokes Shift

- λ_em > λ_ex（發射波長較長）
- 原因：激發後 S₁ 內 vibrational relaxation 以熱耗散能量

### Fluorescence spectrometer

**兩個 monochromator**（excitation + emission），sample 常 90° 放置減少散射干擾

### Fluorescence 比 UV-Vis 靈敏（**Q4 考題**）

10³–10⁶ 倍，因為在近零背景測絕對發光 vs UV-Vis 測 I₀/I 比值

### Quantum dot（量子侷限）

粒徑越小 → 能階越寬 → 藍移
CdSe QD 480–640 nm

## 9. X-ray Diffraction (L5)

### Bragg's Law（必背）

```
n λ = 2d sin θ
```

- **θ 是入射光與晶面的夾角**，不是法線
- d：interplanar spacing
- n：繞射級數
- 建設性干涉條件

### XRD 提供資訊

1. Phase identification（TiO₂ anatase/rutile/brookite）
2. Crystallinity（晶/非晶）
3. Lattice parameters
4. Crystallite size（Scherrer）
5. Residual stress

### Lab XRD 光源：Sealed X-ray tube

- W 鎢絲陰極 → 40 kV 電子 → 金屬靶（Cu, Cr, Fe, Co, Mo）
- **Characteristic Kα + Bremsstrahlung continuum**
- 99% 熱 → 水冷

### Anode 選擇

- **Cu Kα₁ = 1.54060 Å**：無機首選，**但測 Fe/Co 會 fluorescence**
- Co Kα₁ = 1.78897 Å：**測 Fe 用**（Co 能量 < Fe K edge）
- Mo Kα₁ = 0.70930 Å：短波長，小 unit cell

### Sample vs Pattern

| Sample         | Pattern                             |
| -------------- | ----------------------------------- |
| Single crystal | 離散亮點                            |
| Powder（多晶） | 同心圓環（Debye-Scherrer）→ 1D 是峰 |
| Amorphous/液體 | 寬包絡                              |
| Monatomic gas  | 單調背景                            |

### Miller indices (hkl)

晶面在 a,b,c 軸截距的倒數，取整數

- Cubic: 1/d² = (h²+k²+l²)/a²

### 7 晶系 + 14 Bravais lattices

Cubic (P,I,F)、Tetragonal (P,I)、Orthorhombic (P,I,C,F)、Hexagonal、Trigonal、Monoclinic、Triclinic

### Rietveld refinement

Least-squares 擬合整條理論譜 vs 實驗譜，取得精確晶格參數、原子位置、相含量

### Scherrer equation（考題可能考）

```
τ = K λ / (β cos θ)
```

- **K ≈ 0.9**（shape factor）
- **β = FWHM**
- **晶粒越小 → 峰越寬**（估算 nm 尺度）

### Lab XRD vs Synchrotron XRD

Synchrotron 優勢：flux 強、高空間解析度、高角度資訊、快採集、非常規環境、可調能量

## 10. 四題 Exercise 解答快查

### Q1 (L3): aromatic C–H vs C–C 哪個波數高？

**C–H 較高**（~3000 vs ~1500 cm⁻¹）。因為 μ_CH ≪ μ_CC（H 原子 1 vs C 原子 12），由 v̄ ∝ 1/√μ 決定；力常數的差異相對小。

### Q2 (L3 + L4): Grating IR vs FTIR 差別

- Grating：monochromator 一次一波長掃瞄
- FTIR：Michelson interferometer 同時量全頻，再 FFT。更快、更高 SNR、更高解析度

### Q3 (L3): 哪個 detector 可同時用於 IR 與 UV-Vis-NIR？

**InGaAs (800–2500 nm)** 或 **PbS (1000–3500 nm)**，兩者都跨 near-IR 區

### Q4 (L4): Fluorescence vs 激發波長哪個長？

**Emission 較長**（紅移，Stokes shift），因為激發後 S₁ 內的 vibrational relaxation 以熱耗能

### Q5 (L4): XAS absorption edge 物理過程？為什麼元素專一？XANES vs EXAFS？

- 物理：X 光踢出 core electron（photoelectric）→ edge
- 元素專一：每元素 core binding energy 唯一（核電荷決定）
- XANES：edge 近區，給**氧化態、對稱性、d 填充**
- EXAFS：edge 遠區，給**鍵長、配位數、鄰近原子**

### Q6 (L4): Fluorescence vs Phosphorescence (lifetime + spin)

- Lifetime：F ns / P ms–h
- Spin：F 是 S₁→S₀（均 singlet，spin-allowed）；P 是 T₁→S₀（triplet→singlet，需 ISC + spin flip，spin-forbidden）

### Q7 (L4): Fluorescence vs UV-Vis 靈敏度

Fluorescence 靈敏 10³–10⁶ 倍。在近零背景測絕對發光，而 UV-Vis 測兩個大信號的比值（I₀/I），稀樣品時 I≈I₀ 靈敏度被雜訊吃掉。

## 12. 考前最後警示（容易搞混的重點）

1. **「CO₂ 為什麼只有 2 條 IR 峰」**：4 個模式，對稱伸縮 inactive + 兩個彎曲 degenerate
2. **「測 Fe 樣品為何不能用 Cu 靶」**：Cu Kα 能量高於 Fe K edge → 激發 Fe 螢光 → 高背景
3. **「K edge 為何看不清楚 3d」**：1s→3d 是 dipole-forbidden（Δl=2），只能很弱的 quadrupole
4. **「為何 L edge 對 3d 過渡金屬敏感」**：2p→3d 是 dipole-allowed（Δl=1）
5. **「Stokes shift 為什麼是紅移」**：S₁ 內 vibrational relaxation 以熱耗能，發射的能量較小
6. **「Phosphorescence 為何那麼慢」**：T₁→S₀ 需 spin flip，spin-forbidden
7. **「DTGS vs MCT 何時用哪個」**：DTGS 靜態定量高通量，MCT 動力學低通量高靈敏需液氮
8. **「MCT 在高通量會怎樣」**：飽和非線性 → 用 neutral density filter
9. **「為何 quartz cuvette 是 UV 唯一選擇」**：玻璃與塑膠在 UV 會自己吸收
10. **「Scherrer 告訴我們什麼」**：晶粒越小 → 峰越寬
11. **「Rietveld refinement 是在做什麼」**：整條譜的 least-squares 擬合精修結構
12. **「FTIR 三大優勢名詞」**：Fellgett / Jacquinot / Connes
13. **「XAS 元素專一性來自哪」**：每元素的 core electron binding energy 獨一無二
14. **「Synchrotron 比 X-ray tube 強多少」**：10⁶ 倍
15. **「Vacuum UV 為何要抽真空」**：O₂ 強吸收 < 180 nm
16. **「H₂O 有幾個 IR 峰」**：3（3N−6 = 3，全部 IR active）
17. **「為什麼 aromatic C–H 波數高於 aromatic C–C」**：μ_CH ≪ μ_CC
