## Q1. 比較 ATR 與 DRIFT（IR 取樣技術）

**定位同類**：兩者都是 Fourier-Transform IR (FTIR) 光譜儀常用的**樣品取樣技術**，用來測量固體或液體樣品的 IR 光譜，無需傳統的透射壓片法。

**核心差別（一句話）**：IR 光與樣品的交互作用方式不同 — **ATR 利用全反射**，**DRIFT 利用漫反射**。

**物理機制**：
- **ATR (Attenuated Total Reflectance)**：IR 光進入高折射率晶體（如 Diamond、Ge、ZnSe，n > 2.4），在晶體/樣品界面**全反射**。全反射點產生 **evanescent wave（消逝波）** 穿透樣品約 **1 μm**。樣品對特定 IR 頻率吸收 → 反射光減弱 → 量到吸收光譜。穿透深度公式：d_p = λ / (2π·n_c·√(sin²θ − (n_s/n_c)²))
- **DRIFT (Diffuse Reflectance IR FT Spectroscopy)**：IR 光打到粉末樣品的粗糙表面，在顆粒內部**多次散射**，產生 **diffuse reflected light**（漫反射光）與 specular reflected light（鏡面反射光）。偵測 diffuse 部分，光譜反映樣品的**bulk（塊體）特性**。

**適用樣品**：
- **ATR**：流體、膏狀、柔軟固體、聚合物薄膜、液體、生物樣品；**幾乎不需樣品製備**（直接壓在晶體上即可）
- **DRIFT**：粉末、催化劑、粗糙表面固體、錠片；需**磨細**樣品，常以 KBr 稀釋提升訊號

**具體範例**：
- ATR：測聚乙烯薄膜表面化學變化、蛋白質溶液、化妝品
- DRIFT：測粉末狀金屬氧化物催化劑、顏料色粉、礦物分析

## Q2. 比較 Fluorescence 與 Phosphorescence

**定位同類**：兩者都是**光致發光（Photoluminescence）** 的形式，都是分子吸收光子後以光形式發射能量。

**核心差別（一句話）**：**螢光是即時的（ns 級），磷光是延遲的（ms–小時級）**；差別來自**電子自旋狀態**與**躍遷是否被自旋禁止**。

**物理機制**：
- **Fluorescence**：分子吸收光 → 電子躍遷到**激發單重態 S₁**（自旋不變）→ 經 vibrational relaxation 降到 S₁ 最低振動態 → **直接 S₁ → S₀ 發光**。此躍遷是 **spin-allowed**（singlet → singlet，ΔS = 0），所以速率常數 k_f ≈ 10⁶–10⁹ s⁻¹，壽命 ~奈秒（ns）。
- **Phosphorescence**：分子吸收光到 S₁ → 經 **Intersystem Crossing (ISC)** 自旋翻轉進入**激發三重態 T₁** → **T₁ → S₀ 發光**。此躍遷是 **spin-forbidden**（triplet → singlet，需 spin flip），速率常數 k_p ≈ 10⁻¹–10³ s⁻¹，壽命 ms 到小時。

**為什麼磷光比螢光少見？**
- k_IC（Internal Conversion）> k_ISC → 多數分子走 fluorescence 路徑
- **重原子效應**（Br、I、Pt、Pd、Ir）增強 spin-orbit coupling → 提高 k_ISC → 磷光顯著
- 所以 OLED 與夜光材料常用重金屬錯合物

**具體範例**：
- Fluorescence：**螢光筆**、GFP（綠螢光蛋白）、fluorescein 染料
- Phosphorescence：**緊急出口 EXIT 指示牌**、夜光錶、OLED Ir(III) 磷光分子

## Q3. 比較 FTIR 與 Dispersive (Grating) IR

**定位同類**：兩者都是 IR 吸收光譜儀，測量分子振動吸收。

**核心差別（一句話）**：**波長選擇方式不同** — grating IR 用 monochromator **逐波長掃瞄**，FTIR 用 Michelson interferometer **同時量所有頻率**，再經 Fourier Transform 還原成光譜。

**物理機制**：
- **Dispersive IR**：grating monochromator + slit，一次只讓一個波長通過樣品到偵測器，連續旋轉 grating 掃完整個 IR 區段。慢且 SNR 差，因為大部分時間偵測器在閒置。
- **FTIR**：**Michelson interferometer** 有 beam splitter 將光分成兩路（固定鏡 + 動鏡），動鏡移動造成**光程差 δ = 2Δ**，兩光束合併時產生干涉。所有波長的干涉同時被偵測器記錄為 **interferogram I(δ)**。數學上：I(δ) = ∫ B(v̄) cos(2π δ v̄) dv̄，經 **FFT** 反變換得到光譜 B(v̄)。解析度 **Δv̄ = 1/δ_max**（動鏡走越遠解析度越高）。

**FTIR 三大優勢（必背）**：
1. **Fellgett (Multiplex) advantage**：所有頻率同時量 → SNR 提升 √N
2. **Jacquinot (Throughput) advantage**：沒有 slit → 光通量大
3. **Connes advantage**：He-Ne laser 校正動鏡位置 → 波數準確度 ~0.01 cm⁻¹

**具體比較**：
| 項目 | Dispersive | FTIR |
|---|---|---|
| 速度 | 分鐘 | **秒** |
| SNR | 差 | **好** |
| 解析度 | 較差 | **較好** |

**具體範例**：現代分析儀器幾乎都是 FTIR；dispersive IR 已停產，僅存於老舊實驗室。

## Q4. 比較 XANES 與 EXAFS

**定位同類**：兩者都是 **X-ray Absorption Spectroscopy (XAS)** 的子區段，都是在掃瞄 X 光能量通過元素的 absorption edge 時量測吸收係數變化。

**核心差別（一句話）**：能量區段不同 — **XANES** 在 edge 附近（±10–40 eV），提供**電子結構資訊**；**EXAFS** 在 edge 遠端（+40 eV 以上），提供**幾何結構資訊**。

**物理機制**：
- **XANES (X-ray Absorption Near Edge Structure)**：光電子動能低（k 小），**多重散射主導**，反映 core electron 被激發到 bound states 或 continuum 附近。譜形對 d 軌道填充、配位對稱性（Td vs Oh vs square planar）敏感。
- **EXAFS (Extended X-ray Absorption Fine Structure)**：光電子動能高（k 大），**單散射近似可用**。光電子波被鄰近原子散射回吸收原子，發生建設性/破壞性干涉 → μ(E) 出現振盪。振盪的頻率對應鍵長 R，振幅對應配位數 N 與 Debye-Waller factor σ²。公式：χ(k) = −(S₀²/k) Σ [N|f|/R²] e^(−k²σ²) e^(−2R/λ) sin(2kR + φ)。

**提供資訊**：
- **XANES**：氧化態、d 軌道填充、局部對稱性
- **EXAFS**：鍵長、配位數、鄰近原子種類、局部無序度

**具體範例**：
- XANES：**Mn K-edge** 由 6546 eV (MnO, Mn²⁺) → 6549 (Mn₂O₃) → 6552 (MnO₂)，edge 位置正比於氧化態
- EXAFS：量 **Cu-O 鍵長**、催化劑活性中心第一、第二殼層配位數

## Q5. 比較 DTGS 與 MCT Detector（FTIR 偵測器）

**定位同類**：兩者都是 FTIR 最常用的偵測器，將紅外光轉換為電訊號。

**核心差別（一句話）**：DTGS 是 **熱型（thermal / pyroelectric）**，MCT 是 **量子型（photon / quantum）**；兩者適合不同的實驗情境。

**物理機制**：
- **DTGS (Deuterated Triglycine Sulfate)**：**焦電效應**，受熱時材料的自發極化（spontaneous polarization）改變，產生表面電荷變化。Curie 溫度 T_C ≈ 60°C，可**室溫操作**。響應與資料收集速度成反比（S ∝ 1/speed）→ 慢測量訊號好。
- **MCT (HgCdTe, 汞鎘碲)**：**量子效應**，入射光子能量 > band gap 時激發電子從價帶到導帶 → 導電率改變。Band gap 由 Hg₁₋ₓCdₓTe 組成的 x 值調控。需**液氮冷卻 77 K** 降低熱雜訊。響應速度快，信號不隨採集速度而變。

**Detectivity 比較**：
- **MCT-A D\* ≈ 6.4 × 10¹⁰** cm·Hz^(1/2)·W⁻¹
- **DLaTGS D\* ≈ 2.7 × 10⁸**
- **MCT 比 DTGS 靈敏 200 倍**，但 MCT 在**高通量會飽和**非線性，需用 neutral density filter 降光。

**適用情境**：
- **DTGS**：**靜態測量**、定性定量 FTIR、高通量、要寬動態範圍（室溫操作簡便、線性好）
- **MCT**：**動力學測量（kinetics）**、低通量、表面靈敏實驗、FTIR microscopy、快速時間解析（需要高靈敏度與快響應）

**具體範例**：
- DTGS：教學實驗室標準 FTIR、定量分析
- MCT：in situ 催化反應追蹤、FTIR microspectroscopy、synchrotron IR

## Q6. 比較 Prism 與 Grating Monochromator

**定位同類**：兩者都是**波長選擇器**（monochromator），把寬頻光源分散成不同波長後用 slit 挑選出特定波段。

**核心差別（一句話）**：色散物理機制不同 — **prism 用折射（折射率隨波長變化）**，**grating 用繞射**。

**物理機制**：
- **Prism**：光通過稜鏡時，不同波長折射率不同 → 折射角不同 → 色散。短波長（藍/UV）折射角大，長波長（紅/IR）折射角小。**色散非線性**（dn/dλ 不是常數）。
- **Grating**：刻有數千條等距溝槽的反射鏡面，光反射時各溝槽發射子波互相干涉 → 不同波長在不同角度出現建設性極大。遵循 grating equation：**d(sin θ_i + sin θ_r) = mλ**。**色散線性且均勻**。

**優劣比較**：
| 項目 | Prism | Grating |
|---|---|---|
| 色散線性 | 非線性 | **線性** |
| 解析度 | 一般 | **高**（高密度溝槽 → 高解析度）|
| Stray light | **少** | 較多 |
| 現代使用 | 少（老派）| **主流** |

**具體範例**：現代 UV-Vis 與 IR 光譜儀幾乎都用 grating monochromator（全像 holographic grating）；prism 只在某些高精密特殊應用或舊儀器中出現。

## Q7. 比較 K-edge 與 L-edge XAS

**定位同類**：兩者都是 X-ray absorption spectroscopy 的 absorption edge，對應不同主量子數 n 的 core electron 被激發。

**核心差別（一句話）**：激發的電子殼層不同 — **K-edge 是 1s 電子（n=1）**，**L-edge 是 2s / 2p 電子（n=2）**；兩者適合研究不同類型的化學資訊。

**物理機制**：
- **K-edge**：1s 電子被 X 光踢出。能量在 **hard X-ray 區**（>4 keV）。**主 dipole 躍遷是 1s → p**（Δl = ±1 允許）；1s → 3d 是 **dipole-forbidden**（Δl = 2），只能 quadrupole（小 pre-edge peak）。穿透深、可做環境大氣下 in situ 測量。
- **L-edge**：L₁ = 2s, L₂ = 2p₁/₂, L₃ = 2p₃/₂（spin-orbit coupling 分裂）。能量在 **soft X-ray 區**（0.1–2 keV）。**2p → 3d 是 dipole-allowed**（Δl = 1），對過渡金屬 3d 軌道填充極敏感。峰通常比 K-edge 尖銳（稱為 "white lines"）。需 **UHV 環境**（soft X-ray 被空氣吸收）。

**提供資訊**：
- **K-edge**：氧化態、整體幾何對稱性、適合做 EXAFS 分析（鍵長、配位數）
- **L-edge**：**3d 軌道佔據、自旋態（high-spin vs low-spin）、磁性、金屬-配位子共價性**

**具體範例**：
- K-edge：**Mn K-edge** 區分 Mn²⁺/Mn³⁺/Mn⁴⁺ 氧化態
- L-edge：**Fe L₂,₃-edge** 區分 low-spin Fe²⁺（t₂g⁶eg⁰，L₃ 為單峰）vs high-spin Fe³⁺（t₂g³eg²，L₃ 分裂為兩峰）

## Q8. 比較 Single Beam 與 Double Beam Spectrophotometer

**定位同類**：兩者都是 UV-Vis 吸收光譜儀的光學架構。

**核心差別（一句話）**：光路數量不同 — **single beam 只有一條光路需分次量 blank 與 sample**，**double beam 有兩條光路可同時量兩者**。

**物理機制**：
- **Single beam**：光路 Source → Monochromator → Sample → Detector。先放 blank（reference solvent）量 I₀，再換上樣品量 I，計算 A = log(I₀/I)。簡單便宜但**易受光源漂移影響**。
- **Double beam**：光源經 monochromator 後用 **rotating mirror / chopper** 將光以固定頻率（幾十 Hz）交替切到 reference 與 sample 光路。兩條路的訊號分別被偵測，**比率 I_sample/I_reference 即時計算**。偵測器用 lock-in amplifier 鎖定 chopper 頻率。消除光源漂移與偵測器長時漂移。

**優劣**：
- Single：便宜、簡單；缺點是**光源強度漂移直接進入結果**
- Double：貴、複雜；優點是**自動補償漂移、結果穩定、適合動力學或長時間掃瞄**

**具體範例**：
- Single beam：學生實驗室基本 UV-Vis、field instrument
- Double beam：研究級儀器、藥物分析、動力學測量

## Q9. 比較 Filter 與 Monochromator（波長選擇器）

**定位同類**：兩者都用來從寬頻光源挑選出所需波段，是光譜儀的「波長選擇器」。

**核心差別（一句話）**：**Filter 是固定式硬體**（特定波段），**Monochromator 可電子調諧掃瞄全譜**。

**物理機制**：
- **Filter**：
  - **Absorption filter**：用有色玻璃/染料/塑膠吸收不要的波長 → 便宜，但頻寬寬、透射低
  - **Interference filter**（dichroic filter）：薄膜多層結構，用建設性/破壞性干涉讓特定波長通過 → 頻寬可到 1–10 nm，透射率高，但**對入射角度敏感**
- **Monochromator**：prism 或 grating 色散 → slit 挑選 → 可連續調諧。

**優劣**：
| 項目 | Filter | Monochromator |
|---|---|---|
| 彈性 | 低（換波段要換 filter）| **高**（電子調諧）|
| 靈敏度 | **較高**（光損失少）| 較低（slit/鏡面損耗）|
| 成本 | 便宜 | 貴 |
| 解析度 | 固定 | 可調 |

**具體範例**：
- Filter：field instrument、例行比色分析、螢光顯微鏡
- Monochromator：研究級 UV-Vis、IR、XAS

## Q10. 比較 Single Crystal 與 Powder XRD

**定位同類**：兩者都是 X-ray diffraction 測量結晶材料的方法，都遵循 Bragg's Law (nλ = 2d sinθ)。

**核心差別（一句話）**：樣品形態不同 — **單晶方向固定 → 看到少數離散亮點**；**粉末有眾多隨機取向微晶 → 看到同心圓環**。

**物理機制**：
- **Single Crystal**：一個完整晶體取向固定 → 只有特定 (hkl) 晶面在特定 θ 滿足 Bragg 條件 → **偵測器上看到數個離散亮點**。每個亮點對應一組晶面。
- **Powder (polycrystalline)**：數百萬微晶隨機取向 → 每組晶面在**各個角度**都有晶粒處於 Bragg 條件 → 亮點沿 2θ 旋轉對稱分布形成**同心圓環（Debye-Scherrer rings）**。在一維 θ-2θ 掃瞄下變成峰。

**資訊提取**：
- Single crystal：完整 3D 結構（原子位置、鍵結）
- Powder：相鑑定（phase ID）、晶格參數、**晶粒大小（Scherrer equation）**、相含量（Rietveld refinement）

**非晶 vs 氣體對照**：
- **Amorphous/液體**：寬而漫的包絡（broad hump）
- **單原子氣體**：只有單調衰減背景

**具體範例**：
- Single crystal：蛋白質結構（protein crystallography）、novel 化合物
- Powder：**TiO₂ anatase 與 rutile 相鑑定**、奈米粒子晶粒大小

## Q11. 比較 Hard / Tender / Soft X-ray XAS

**定位同類**：三者都是 XAS 的能量分類，決定可測的元素範圍與實驗環境。

**核心差別（一句話）**：X 光能量越低，穿透能力越弱但對輕元素電子結構越敏感。

**物理機制**：
| 類別 | 能量 | 測什麼 | 環境 |
|---|---|---|---|
| **Hard XAS** | > 5 keV | 過渡金屬 K-edge (Fe, Co, Ni...)、重元素 L-edge | 大氣下可 in situ |
| **Tender XAS** | 2–5 keV | P, S, Cl K-edge | 需 **He-purged**（N₂/O₂ 會吸收）|
| **Soft XAS** | < 2 keV | C/N/O K-edge、3d 過渡金屬 **L-edge** | 需 **UHV** |

**為什麼環境要求不同**：
- Hard X-ray 穿透空氣、塑膠、水 → 便利
- Soft X-ray 被空氣強吸收（幾 cm 就擋光） → 必須超高真空
- Tender 區夾在中間，用 He 取代 N₂/O₂ 可行

**具體範例**：
- Hard：Fe、Cu 催化劑的 K-edge in situ 電化學測量
- Tender：生物樣品 S、P K-edge
- Soft：有機分子 C, N, O K-edge、過渡金屬 L-edge 磁性研究

## Q12. 比較 UV-Vis Absorption 與 Fluorescence Spectroscopy（靈敏度）

**定位同類**：兩者都是用 UV-Vis 光激發分子的光譜技術。

**核心差別（一句話）**：UV-Vis 量**吸收**（光透射的衰減），Fluorescence 量**發射**（激發後發出的光）；**Fluorescence 對稀樣品靈敏 10³–10⁶ 倍**。

**物理機制**：
- **UV-Vis absorption**：A = log(I₀/I)，需要測量**透射強度與入射強度的比值**。當樣品很稀時，I ≈ I₀，log(I₀/I) → 0，而**儀器雜訊本身就有一定大小** → 小的吸收訊號被雜訊淹沒 → 靈敏度差。
- **Fluorescence**：測量樣品**在 90° 方向發射的光絕對量**。沒有樣品時背景接近 0，任何微量螢光分子都會產生高於背景的信號 → 雜訊較小、信號-背景比極高 → 靈敏度極高。

**靈敏度差距**：
- UV-Vis absorption 實用下限 ≈ ppm（A 下限 ~0.001）
- Fluorescence 實用下限 ≈ **ppb 到 ppt**
- **差距 10³–10⁶ 倍**

**儀器對應**：
- UV-Vis：一個 monochromator + 透射光路
- Fluorescence：**兩個 monochromator**（excitation + emission），sample 放 90° 減少散射

**具體範例**：
- UV-Vis：定量中高濃度溶液（藥物、染料）
- Fluorescence：痕量偵測生物分子、GFP 蛋白、單分子螢光、ppb 嗎啡偵測

## Q13. 比較 In-house XRD 與 Synchrotron XRD

**定位同類**：兩者都是 X 光繞射實驗設施。

**核心差別（一句話）**：光源強度與性質不同 — **in-house 用 sealed X-ray tube**（實驗室桌上型），**synchrotron 用高能電子束在磁鐵中發射同步輻射**（10⁶ 倍強）。

**物理機制**：
- **In-house**：鎢絲陰極 + 金屬靶（Cu/Mo/Co/Cr/Fe）+ 40 kV 加速電壓 → 電子撞擊產生 characteristic X-ray（Kα/Kβ）+ Bremsstrahlung 連續背景。99% 能量變熱，需水冷。Cu Kα₁ = 1.54060 Å 為最常用。
- **Synchrotron**：電子儲存環（如 TPS 3 GeV、TLS 1.5 GeV）中 GeV 級電子在彎轉磁鐵、wiggler 或 undulator 中改變方向時發射 synchrotron radiation，覆蓋從 IR 到 hard X-ray 連續頻譜，可用 monochromator 挑選單色光。

**Synchrotron 優勢**：
1. **Intense photon flux** —— 比 X-ray tube **強 10⁶ 倍**
2. **High spatial resolution** —— 可做 micro-diffraction
3. **High angle information** —— 大 2θ 角峰資訊豐富
4. **Fast data acquisition** —— 20 秒 vs 30 分鐘
5. **Non-ambient conditions** —— 適合 in situ（高溫、高壓、電化學）
6. **Tunable energy** —— 可挑任意波長做 resonance 實驗

**具體範例**：
- In-house：日常相鑑定、化學系課程研究
- Synchrotron：**TPS（Taiwan Photon Source, 2016, 3 GeV）**、SPring-8、APS；做 in situ 電池充放電、time-resolved XRD

## Q14. 比較 Thermal Detector 與 Photon (Quantum) Detector（IR 偵測器兩大類）

**定位同類**：兩者都是 IR 偵測器，但原理完全不同。

**核心差別（一句話）**：**Thermal detector 測輻射造成的「溫度變化」**；**Photon detector 測光子直接「激發電子」的量子效應**。

**物理機制**：
- **Thermal detectors**：
  - Thermocouple/thermopile：熱電效應（兩金屬溫差產生 EMF）
  - Bolometer：金屬或半導體熱敏電阻
  - **Pyroelectric (DTGS)**：電介質自發極化隨 T 變化（需 Curie 溫度）
  - Golay cell：氣體熱膨脹使膜變形
  - **共同特徵**：響應慢（ms–s 級）、對整個 IR 寬頻響應、**室溫可操作**、信號正比於入射功率
- **Photon detectors**：
  - MCT (HgCdTe)、InSb、PbSe、InGaAs、Ge 等窄 bandgap 半導體
  - 入射光子能量 ≥ bandgap → 電子從價帶跳到導帶 → 導電率改變
  - **共同特徵**：響應快（~ns）、靈敏度高、SNR 好、但**需液氮 77 K 冷卻**減少熱雜訊、有波長上限（由 bandgap 決定）

**優劣**：
| 項目 | Thermal | Photon |
|---|---|---|
| 響應速度 | 慢 | **快** |
| 靈敏度 | 中 | **高** |
| 操作溫度 | 室溫 | 液氮 77 K |
| 波長範圍 | 廣 | 受 bandgap 限制 |

**具體範例**：
- Thermal：DTGS 在定量 FTIR、Bolometer 在 far-IR
- Photon：MCT 在 FTIR kinetics、InGaAs 在 near-IR、Doped Ge 在 far-IR 天文觀測

## Q15. 比較 Homonuclear 與 Heteronuclear Diatomic 分子 的 IR Activity

**定位同類**：兩者都是雙原子分子（diatomic molecules），只有一個振動模式（stretch）。

**核心差別（一句話）**：**Heteronuclear（兩個不同原子）有永久電偶極矩 → IR active**；**homonuclear（兩個相同原子）永遠無電偶極矩 → IR inactive**。

**物理機制**：
IR 吸收的必要條件是 **dipole moment μ 在振動過程中必須改變**。μ = Q·r（電荷 × 向量距離）。

- **Heteronuclear（HCl、CO、HF）**：
  - 兩原子電負度不同 → 電子分布不均 → 分子**有永久 dipole**
  - 振動時 r 變化 → μ 大小改變 → **IR active**
  - 例：HCl 在 2890 cm⁻¹ 有強吸收
- **Homonuclear（H₂、N₂、O₂、Cl₂）**：
  - 兩原子相同 → 電負度相同 → 電子均勻分布 → **μ = 0**
  - 振動時電荷分布仍然對稱 → μ 始終為 0 → **μ 不改變** → **IR inactive**
  - 例：N₂ 的 stretching（雖然頻率 ~2330 cm⁻¹）**完全看不到**

**大氣含義**：
- 主要大氣組成 N₂、O₂ 是 **homonuclear 分子 → 不吸收 IR** → 大氣對 IR 大致透明
- 少量的 **H₂O、CO₂、CH₄**（heteronuclear 或多原子）吸收 IR → 是**溫室氣體**，造成地球溫度升高

**擴展推論**：
- 這也是為什麼**紅外天文學可以透過大氣觀測**（雖有 H₂O、CO₂ 干擾帶）
- 為什麼 Raman spectroscopy 對 homonuclear 分子反而有效（Raman 規則是 polarizability 變化，不是 dipole 變化）

**具體範例**：
- IR active：HCl、CO、HF、C=O 羰基、O-H
- IR inactive：H₂、N₂、O₂、Cl₂

## 📝 答題技巧總結

### 考試時套用這個結構：
```
1. [定位同類] — 一句話說兩者是什麼大類技術
2. [一句話點出差別] — 核心 distinction
3. [物理機制] — 分別解釋兩者的原理，包含
   - 關鍵公式（如 Bragg、Beer-Lambert、Hooke、FFT）
   - 關鍵物理量（波長、能量、折射率、dipole）
4. [適用情境 / 優劣] — 什麼時候該用哪個
5. [具體範例] — 給一個可驗證的實際例子（儀器名稱、樣品類型、應用領域）
```

- [ ] 有沒有點出「核心差別」的那一句話？

- [ ] 有沒有一個**具體範例**？
