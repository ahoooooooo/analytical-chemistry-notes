## 🔢 必背公式（5 分鐘）

```
# 光的基本
c = λν                             (光速 = 波長 × 頻率)
v̄ = 1/λ (cm⁻¹)                     (波數)
E = hν = hc/λ = hc·v̄                (光子能量)
λ(nm) × E(eV) = 1240               ← 超好用捷徑

# 吸收光譜（UV-Vis、IR、XAS 都用）
A = log(I₀/I) = −log T = εbc        (Beer-Lambert)
μx = ln(I₀/I_t)                     (XAS)

# IR 振動
v̄ = (1/2πc)√(f/μ)                  (Hooke's Law)
μ = M₁M₂/(M₁+M₂)                    (reduced mass)
非線型：3N−6；線型：3N−5 振動模式

# X 光繞射
nλ = 2d sinθ                        (Bragg's Law)
τ = Kλ/(β cosθ)                     (Scherrer，晶粒大小)
1/d² = (h²+k²+l²)/a²                (Cubic d-spacing)

# FTIR
Δv̄ = 1/δ_max                       (解析度 = 1/動鏡最大位移)

# SNR
(S/N)_n = √n × (S/N)₁              (平均 n 次提升 √n)
```

## 🎯 選擇定則（必背 3 分鐘）

**IR**：分子振動時 **dipole moment 必須改變**
- Homonuclear（H₂、N₂、O₂、Cl₂）→ 永遠 IR inactive
- Heteronuclear（HCl、CO）→ IR active
- **CO₂ 對稱伸縮 IR inactive**，但反對稱伸縮與彎曲 active → 共 4 個模式但只看到 2 條帶
- **H₂O 3 個模式全部 IR active** → 看到 3 條帶

**UV-Vis**：**ΔS = 0**（自旋不變）
- Singlet → Singlet 允許
- Singlet → Triplet 禁止
- High-spin d⁵：所有 d-d 都改變 spin → 顏色很淡

**XAS (dipole)**：Δl = ±1
- K-edge: 1s → p
- L-edge: 2s → p，**2p → 3d 允許**
- **K-edge 1s → 3d 禁止**（只能 quadrupole 弱 pre-edge）
- **L₂,₃ edge 對 3d 過渡金屬更敏感**

## 🏆 最可能的 5 大比較題（20 分鐘重點）

### Q1. Fluorescence vs Phosphorescence

|  | Fluorescence | Phosphorescence |
|---|---|---|
| 能階 | S₁ → S₀（singlet） | T₁ → S₀（triplet→singlet） |
| 自旋 | 不變（allowed） | 需翻轉（forbidden） |
| 速率 | ns（10⁷–10¹⁰ s⁻¹） | ms–小時（10⁻³–10⁵ s⁻¹） |
| 移光源後 | 立刻停 | 持續發光 |
| 範例 | **螢光筆、GFP** | **EXIT 指示牌、夜光** |

**為什麼磷光少見**：k_IC > k_ISC，ISC 是禁止的 → 多數分子走螢光路徑。重原子（Br、I、Pt、Ir）增強 spin-orbit coupling → 磷光顯著。

---

### Q2. ATR vs DRIFT

|  | **ATR** | **DRIFT** |
|---|---|---|
| 機制 | **全反射 + evanescent wave** | 漫反射 |
| 穿透深度 | ~1 μm（表面） | **bulk（塊體）** |
| 晶體 | Diamond、Ge、ZnSe（n>2.4） | 粉末 + KBr |
| 樣品 | 液體、膜、膏狀、免製備 | 粉末、磨細、KBr 稀釋 |
| 公式 | d_p = λ/(2πn_c√(sin²θ−(n_s/n_c)²)) | — |

**口訣**：ATR = 表面 / 全反射 / 免壓片；DRIFT = 內部 / 漫反射 / 粉末

---

### Q3. FTIR vs Dispersive IR

|  | Dispersive | **FTIR** |
|---|---|---|
| 光學 | Monochromator + slit | **Michelson interferometer** |
| 量測 | 一次一波長（慢）| **同時全頻段** |
| 速度 | 分鐘 | 秒 |
| SNR | 差 | **好** |

**FTIR 三大優勢**：
1. **Fellgett**：全頻段同時 → SNR √N
2. **Jacquinot**：無 slit → 光通量大
3. **Connes**：He-Ne laser 校正波數

**關鍵公式**：Δv̄ = 1/δ_max；I(δ) → FFT → B(v̄)

---

### Q4. XANES vs EXAFS

|  | **XANES** | **EXAFS** |
|---|---|---|
| 能量 | edge ±10–40 eV | edge +40 eV 以上 |
| 散射 | 多重散射 | 單散射 |
| 資訊 | **氧化態、d 填充、對稱性** | **鍵長、配位數、鄰近原子** |

**範例**：
- XANES：Mn K-edge 6546→6549→6552 eV 對應 Mn²⁺/³⁺/⁴⁺
- EXAFS：Cu-O 鍵長、催化劑配位數

---

### Q5. DTGS vs MCT (FTIR 偵測器)

|  | **DTGS** | **MCT** |
|---|---|---|
| 類型 | Thermal（焦電）| Quantum（光子）|
| 溫度 | 室溫 | **液氮 77 K** |
| D* | 2.7×10⁸ | **6.4×10¹⁰（高 200 倍）**|
| 高通量 | 線性好 | **容易飽和非線性** |
| 適合 | 靜態、定量 | **動力學、低通量** |

## 🔬 其他關鍵 (10 分鐘)

### UV-Vis 三種電子躍遷
- **π → π\***：不飽和化合物
- **n → π\***：有 lone pair
- **d-d**：過渡金屬，顏色來源

### UV-Vis Cuvette 材質
- **Quartz 170–2700 nm**（UV 必用！）
- Glass 334–2500（不能 UV）
- Plastic 340–800（便宜）

### UV-Vis 光源
- **D₂ lamp**：185–400 UV
- **Tungsten-halogen**：350–3000 Vis-NIR
- **Xenon flash**：185–2500 單一光源全段

### XAS 能量分類
- Soft < 2 keV：輕元素 K、3d 過渡金屬 L，**需 UHV**
- Tender 2–5 keV：P、S，需 **He-purged**
- Hard > 5 keV：過渡金屬 K、重元素，**大氣下可 in situ**

### Bragg's Law 用法
- **θ 是入射光與晶面夾角**，不是法線！
- 2θ 是實驗測量角度
- 記得 θ = 2θ/2

### Scherrer equation 要點
- **晶粒越小 → 峰越寬**
- β 要用 **radian**（度 × π/180）
- K ≈ 0.9

### XRD Cu 靶 vs Fe 樣品
- Cu Kα 能量 > Fe K edge → 激發 Fe X 光螢光 → 高背景
- **測 Fe 改用 Co 或 Cr 靶**

### 同步輻射優勢
- 比 tube 強 10⁶ 倍
- **TPS 3 GeV（2016）**、TLS 1.5 GeV
- 連續寬頻、可調能量、in situ 測量

## ⚠️ 易混淆警示清單（5 分鐘）

1. **CO₂ 2 條帶**：4 個模式，對稱伸縮 inactive + 兩個彎曲 degenerate
2. **測 Fe 不能用 Cu 靶**：會激發 Fe 螢光
3. **K edge 看不清 3d**：1s→3d 是 dipole-forbidden
4. **L edge 對 3d 過渡金屬敏感**：2p→3d dipole-allowed
5. **Stokes shift 紅移**：S₁ 內 vibrational relaxation 耗能
6. **磷光慢**：T₁→S₀ 需 spin flip，forbidden
7. **DTGS 靜態 / MCT 動力學**，MCT 高通量飽和
8. **UV 必用 quartz cuvette**，glass/plastic 在 UV 會自己吸收
9. **晶粒小 → XRD 峰寬**（Scherrer）
10. **H₂O 有 3 條 IR 帶**（全部 active）
11. **Aromatic C-H > C-C**：因為 μ_CH ≪ μ_CC
12. **FTIR 三大優勢**：Fellgett / Jacquinot / Connes
13. **XAS 元素專一性**：core electron binding energy 獨特
14. **Vacuum-UV 需抽真空**：O₂ 強吸收 < 180 nm
15. **Synchrotron > Tube 10⁶ 倍**
16. **螢光對稀樣品比 UV-Vis 靈敏 10³–10⁶ 倍**：近零背景 vs log 比值

## 🎯 記憶口訣速查

- **IR 振動**：鍵強+原子輕 → 波數高
- **Beer-Lambert**：A = εbc，**稀溶液才準**
- **Bragg**：nλ = 2d sinθ，**θ 不是 2θ**
- **Scherrer**：小晶粒 → 寬峰
- **XAS edge**：元素專一 = 核電荷決定
- **FTIR**：Michelson + FFT
- **ATR**：全反射 + evanescent
- **Fluorescence**：singlet 快、**磷光 triplet 慢**
