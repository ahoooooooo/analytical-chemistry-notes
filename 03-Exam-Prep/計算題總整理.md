## 🧮 需要記住的常數（考試前抄一張在計算機上）

```
h  = 6.626 × 10⁻³⁴ J·s          (Planck)
c  = 3.00 × 10⁸ m/s             (光速，真空)
   = 3.00 × 10¹⁰ cm/s           (換 cm/s 計算波數用)
1 eV = 1.602 × 10⁻¹⁹ J
N_A  = 6.022 × 10²³ /mol        (Avogadro)
1 Å  = 10⁻¹⁰ m = 0.1 nm
原子質量單位：1 u = 1.66 × 10⁻²⁷ kg
```

## 題型 1：波長/頻率/波數/能量互換

### 公式
```
c = λν                          → ν = c/λ
v̄ = 1/λ                         (λ 要用 cm)
E = hν = hc/λ = hc·v̄
1 eV = 1.602 × 10⁻¹⁹ J
```

### 【範例 1-1】計算 λ = 500 nm 的波數與能量
**解**：
- λ = 500 nm = 500 × 10⁻⁷ cm = 5 × 10⁻⁵ cm
- **波數**：v̄ = 1/λ = 1/(5 × 10⁻⁵) = **20000 cm⁻¹**
- **頻率**：ν = c/λ = (3 × 10⁸)/(500 × 10⁻⁹) = **6 × 10¹⁴ Hz**
- **能量 (J)**：E = hν = 6.626 × 10⁻³⁴ × 6 × 10¹⁴ = **3.98 × 10⁻¹⁹ J**
- **能量 (eV)**：E = 3.98 × 10⁻¹⁹ / 1.602 × 10⁻¹⁹ ≈ **2.48 eV**

### 【範例 1-2】IR 波長 5.00 μm → 波數 = ?
**解**：
- λ = 5.00 μm = 5.00 × 10⁻⁴ cm
- v̄ = 1/λ = 1/(5.00 × 10⁻⁴) = **2000 cm⁻¹**

### 【範例 1-3】光子能量 2000 cm⁻¹ 等於多少 J？多少 eV？
**解**：
- E = hc·v̄ = (6.626 × 10⁻³⁴)(3 × 10¹⁰)(2000)
- = 3.976 × 10⁻²⁰ J
- ÷ 1.602 × 10⁻¹⁹ = **0.248 eV**

### 速查：常見波數 ↔ 波長 ↔ 能量
| v̄ (cm⁻¹) | λ | E (eV) |
|---|---|---|
| 2000 | 5 μm | 0.25 |
| 4000 | 2.5 μm | 0.50 |
| 25000 | 400 nm | 3.1 |
| 50000 | 200 nm | 6.2 |

## 題型 2：Beer-Lambert Law（定量分析）

### 公式
```
A = −log T = log(P₀/P) = ε·b·c
T = P/P₀
%T × 100 → A = 2 − log(%T)
```

### 【範例 2-1】某溶液 %T = 44.1%（在 b = 2.10 cm，c = 0.003818 M），求 ε
**解**（從 L1 講義 Example 24-5）：
- A = −log T = −log(0.441) = **0.356**
- ε = A/(b·c) = 0.356 / (2.10 × 0.003818)
- = **44.5 L·mol⁻¹·cm⁻¹**（這題算 KMnO₄ 的 ε）

### 【範例 2-2】已知 ε = 7250 L·mol⁻¹·cm⁻¹，b = 1.00 cm，測得 A = 0.513，求濃度
**解**：
- c = A/(εb) = 0.513 / (7250 × 1.00)
- = **7.08 × 10⁻⁵ M**

### 【範例 2-3】濃度翻倍，A 如何變？
**解**：A = εbc，c 翻倍 → **A 翻倍**（線性）

### 【範例 2-4】光程從 1 cm 減半為 0.5 cm，原本 A = 1.20，現在 A = ?
**解**：A = εbc，b 減半 → **A = 0.60**

### 【範例 2-5】稀釋倍數計算
原本 c₁，A₁ = 1.80；稀釋 10 倍 (c₂ = c₁/10)，A₂ = ?
**解**：A₂ = A₁/10 = **0.18**（在 Beer's Law 線性區內）

## 題型 4：Hooke's Law 振動頻率

### 公式
```
v̄ = (1/(2πc)) × √(f/μ)       (v̄ 單位 cm⁻¹，c 要用 cm/s = 3×10¹⁰)
μ = (M₁ M₂) / (M₁ + M₂)       (kg)
```

**注意單位**：
- c 用 **3 × 10¹⁰ cm/s**（不是 m/s！）
- f 用 **N/m**
- M 用 **kg**（記得 g/mol 要除以 N_A，再乘 10⁻³ 換成 kg）
- 原子質量換算：M (kg) = M (g/mol) / (6.022 × 10²³) / 1000 = M (g/mol) × 1.66 × 10⁻²⁷

### 【範例 4-1】C=O 鍵伸縮振動波數（f ≈ 1 × 10³ N/m）
**解**（L2 Example 16-1）：
- M_C = 12 × 1.66 × 10⁻²⁷ = 2.0 × 10⁻²⁶ kg
- M_O = 16 × 1.66 × 10⁻²⁷ = 2.7 × 10⁻²⁶ kg
- μ = (2.0 × 2.7)/(2.0 + 2.7) × 10⁻²⁶ = 1.14 × 10⁻²⁶ kg
- v̄ = (1/(2π × 3×10¹⁰)) × √(1000/1.14×10⁻²⁶)
- = (5.3 × 10⁻¹²) × √(8.77 × 10²⁸)
- = 5.3 × 10⁻¹² × 2.96 × 10¹⁴
- ≈ **1570 cm⁻¹**（接近實驗值 1600-1800 cm⁻¹）

### 【範例 4-2】C–H 與 C–D 的振動頻率比
**解**：
- f 相同（都是 C-H/C-D single bond）
- μ_CH = (12×1)/(12+1) ≈ 0.923 u
- μ_CD = (12×2)/(12+2) ≈ 1.714 u
- **v̄_CD / v̄_CH = √(μ_CH/μ_CD) = √(0.923/1.714) = √0.538 ≈ 0.733**
- 也就是說 C-D 的波數只有 C-H 的 **73%**
- 若 C-H 在 3000 cm⁻¹，則 C-D 約在 **2200 cm⁻¹**

### 【範例 4-3】C≡C vs C=C vs C–C 的波數比
**解**：
- μ 相同（都是 C-C 對）
- f 比例為 3 : 2 : 1
- v̄ 比例 = √3 : √2 : 1 ≈ 1.73 : 1.41 : 1
- 若 C-C 在 1000 cm⁻¹，則 C=C 在 1410，C≡C 在 1730（實驗值 1000/1650/2150，接近）

### 【範例 4-4】Reduced mass 快算
O-H：μ = (16×1)/17 = 0.941 u
C-H：μ = (12×1)/13 = 0.923 u
N-H：μ = (14×1)/15 = 0.933 u
C-O：μ = (12×16)/28 = 6.86 u
C=C：μ = (12×12)/24 = 6.0 u
C-Cl：μ = (12×35.5)/47.5 = 8.97 u

## 題型 5：振動模式數 / 自由度

### 公式
```
非線型分子： 3N − 6 個振動模式
線型分子：   3N − 5 個振動模式
```

### 【範例 5-1】H₂O 的振動模式數
- 非線型，N = 3
- 3×3 − 6 = **3 個**（對稱伸縮、反對稱伸縮、剪式彎曲）
- 全部 IR active

### 【範例 5-2】CO₂ 的振動模式數
- 線型，N = 3
- 3×3 − 5 = **4 個**
- 對稱伸縮 (IR inactive) + 反對稱伸縮 (active) + 兩個 degenerate 彎曲 (active)
- **實際看到 2 條 IR 帶**

### 【範例 5-3】C₆H₆ (苯) 振動模式數
- 非線型，N = 12
- 3×12 − 6 = **30 個**

### 【範例 5-4】NH₃
- 非線型，N = 4
- 3×4 − 6 = **6 個**

### 【範例 5-5】HCl
- 線型，N = 2
- 3×2 − 5 = **1 個**（只有 bond stretch）
- 異核 → **IR active**

## 題型 6：Bragg's Law

### 公式
```
n λ = 2 d sin θ
```
注意：2θ 是實驗測量值（detector 與入射光的夾角），**θ = 2θ/2**

### 【範例 6-1】Cu Kα (λ = 1.5406 Å) 打一晶體，n=1 時測得 2θ = 30°，求 d
**解**：
- θ = 15°
- d = λ / (2 sinθ) = 1.5406 / (2 × sin15°)
- = 1.5406 / (2 × 0.2588)
- = **2.98 Å**

### 【範例 6-2】NaCl (200) 面 d = 2.82 Å，用 Cu Kα，n=1 的 2θ = ?
**解**：
- sinθ = nλ/(2d) = (1×1.5406)/(2×2.82) = 0.2732
- θ = 15.86°
- **2θ = 31.72°**

### 【範例 6-3】Cubic 晶體 a = 5.64 Å（NaCl），(200) 晶面的 d = ?
**解**：
- 1/d² = (h²+k²+l²)/a² = (4+0+0)/5.64² = 4/31.81 = 0.1258
- d² = 7.95 Å²
- d = **2.82 Å**

### 【範例 6-4】若改用 Mo Kα (λ = 0.7093 Å) 測同一 d = 2.98 Å 晶面，2θ = ?
**解**：
- sinθ = 0.7093/(2 × 2.98) = 0.1190
- θ = 6.84°
- **2θ = 13.68°**（比 Cu 小很多 → Mo 適合小 unit cell 高分辨）

### 【範例 6-5】Cubic 晶體 a = 4.00 Å，Cu Kα，(111) 面 2θ = ?
**解**：
- d² = a²/(h²+k²+l²) = 16/3 = 5.333
- d = 2.31 Å
- sinθ = 1.5406/(2 × 2.31) = 0.333
- θ = 19.47°
- **2θ = 38.94°**

## 題型 7：Cubic 系統 d_hkl

### 公式
```
1/d² = (h² + k² + l²) / a²   (cubic)
```

### 【範例 7-1】a = 4.0 Å，列出 (100)、(110)、(111)、(200)、(210) 的 d
**解**：
| (hkl) | h²+k²+l² | d² (Å²) | d (Å) |
|---|---|---|---|
| (100) | 1 | 16 | 4.00 |
| (110) | 2 | 8 | 2.83 |
| (111) | 3 | 5.33 | 2.31 |
| (200) | 4 | 4 | 2.00 |
| (210) | 5 | 3.2 | 1.79 |

### 【範例 7-2】Cu (fcc) a = 3.615 Å，(111) 面的 2θ（用 Cu Kα）
**解**：
- d = a/√3 = 3.615/1.732 = 2.087 Å
- sinθ = 1.5406/(2×2.087) = 0.369
- θ = 21.66°
- **2θ = 43.32°**（這是 Cu (111) 的標準位置）

## 題型 8：Scherrer Equation（晶粒大小）

### 公式
```
τ = K λ / (β cos θ)
```
- K ≈ 0.9
- λ 用 **nm** 或 **Å**（跟 τ 相同單位）
- β = FWHM，**必須用弧度（radian）**（不是度！）
- 換算：度 × π/180 = radian

### 【範例 8-1】Cu Kα (λ = 0.154 nm)，峰在 2θ = 40°，FWHM = 0.5°，求晶粒大小
**解**：
- θ = 20°
- β = 0.5° × π/180 = 0.00873 rad
- τ = (0.9 × 0.154)/(0.00873 × cos20°)
- = 0.1386/(0.00873 × 0.9397)
- = 0.1386/0.00820
- ≈ **16.9 nm**

### 【範例 8-2】同樣條件下，FWHM 變 2 倍（晶粒變多小？）
**解**：β 翻倍 → τ 減半 → **8.45 nm**
- 規則：**FWHM 與晶粒大小成反比**

### 【範例 8-3】峰很銳（FWHM = 0.1°）與峰很寬（FWHM = 2°），τ 比值？
**解**：τ 反比於 β → 比值 20 倍（0.1°的晶粒是 2°的 20 倍大）

## 題型 9：XAS Beer's Law

### 公式
```
I_t = I₀ · exp(−μ(E)·x)
μ(E)·x = ln(I₀/I_t)
```

### 【範例 9-1】I₀ = 1000，I_t = 368，樣品厚度 x = 0.1 mm，求 μ(E)
**解**：
- μx = ln(1000/368) = ln(2.717) ≈ 1.0
- μ = 1.0/0.1 mm = **10 mm⁻¹ = 100 cm⁻¹**

### 【範例 9-2】樣品厚度對 I_t/I₀ 的影響
- 若 μx = 0.5：I_t/I₀ = e⁻⁰·⁵ = 0.607（60.7%）
- 若 μx = 1.0：I_t/I₀ = e⁻¹ = 0.368（36.8%）
- 若 μx = 2.0：I_t/I₀ = e⁻² = 0.135（13.5%）

## 題型 10：ATR Penetration Depth

### 公式
```
d_p = λ / (2π n₁ √(sin²θ − (n₁/n₂)²))   ← 這是講義版本
```
注意：不同教科書符號不一樣，有的 n₁ 是晶體，n₂ 是樣品；有的相反。**考試看哪個高折射率就是晶體**。

### 【範例 10-1】ATR 用 ZnSe (n=2.4)，樣品 n=1.5，入射角 45°，波長 10 μm，求 d_p
**解**：
- sin²45° = 0.5
- (n_sample/n_crystal)² = (1.5/2.4)² = 0.391
- √(0.5 − 0.391) = √0.109 = 0.330
- d_p = 10/(2π × 2.4 × 0.330)
- = 10/4.977
- ≈ **2.01 μm**

### 【範例 10-2】換用 Ge (n=4.0) 同條件
**解**：
- (1.5/4.0)² = 0.1406
- √(0.5 − 0.1406) = √0.3594 = 0.5995
- d_p = 10/(2π × 4.0 × 0.5995) = 10/15.07 ≈ **0.66 μm**
- 高折射率晶體 → **穿透深度更淺**

### 【範例 10-3】ATR 臨界角 (ZnSe, n=2.4; 樣品 n=1.5)
**解**：
- θ_c = sin⁻¹(n_s/n_c) = sin⁻¹(1.5/2.4) = sin⁻¹(0.625)
- = **38.7°**（入射角 > 38.7° 才會全反射）

## 題型 11：Snell's Law（光的折射）

### 公式
```
n₁ sin θ₁ = n₂ sin θ₂
```

### 【範例 11-1】光從空氣 (n=1) 進入玻璃 (n=1.5)，入射角 30°，折射角 = ?
**解**：
- sin θ₂ = (n₁/n₂) × sin θ₁ = (1/1.5) × 0.5 = 0.333
- θ₂ = **19.47°**

## 題型 12：FTIR 解析度

**解**：
- δ_max = 1/0.5 = **2 cm**

## 題型 13：Signal-to-Noise Ratio（Ensemble Averaging）

### 公式
```
(S/N)_n = √n × (S/N)_1
```
n = 掃瞄次數或取樣次數

### 【範例 13-1】單次掃瞄 S/N = 10，要達到 S/N = 100，需要掃幾次？
**解**：
- 100 = √n × 10
- √n = 10
- **n = 100 次**

### 【範例 13-2】掃 16 次比 1 次 S/N 提升多少倍？
**解**：
- √16 = **4 倍**

### 【範例 13-3】S/N 從 5 提升到 20 (4 倍)，需要掃幾次？
**解**：n = 4² = **16 次**

## 題型 14：Tauc Plot（Band Gap）

### 公式
```
(α·hν)^(1/γ) = B (hν − Eg)
γ = 1/2：direct allowed
γ = 2  ：indirect allowed
Amorphous: (α·hν)^(1/2) = B(hν − Eg)
```

### 【範例 14-1】一 amorphous 半導體 Tauc plot (y = (αhν)² vs x = hν) 線性段外插到 x 軸 = 3.22 eV，Eg = ?
**解**：**Eg = 3.22 eV**（x 軸截距就是 Eg）

### 【範例 14-2】TiO₂ anatase Eg = 3.22 eV，對應吸收邊波長 = ?
**解**：
- λ = hc/E = 1240/3.22 ≈ **385 nm**（UV-A 區）
- 記這個捷徑：**λ(nm) = 1240/E(eV)**

### 【範例 14-3】金屬氧化物 Eg = 2.48 eV，會吸收什麼顏色的光？
**解**：λ = 1240/2.48 ≈ 500 nm（綠光）→ 呈現**紅色**（吸收互補色）

## 🎯 考前必熟的快捷換算

```
λ(nm) = 1240 / E(eV)              ← 超好用！
E(eV) = 1240 / λ(nm)
v̄(cm⁻¹) = 10⁷ / λ(nm)
v̄(cm⁻¹) = 10⁴ / λ(μm)
```

### 範例：
- λ = 500 nm → E = 1240/500 = **2.48 eV**
- λ = 2.5 μm → v̄ = 10⁴/2.5 = **4000 cm⁻¹**
- E = 3.1 eV → λ = 1240/3.1 = **400 nm**
- v̄ = 2000 cm⁻¹ → λ = 10⁴/2000 = **5 μm**

## 📋 考試計算策略

1. **先看單位**：90% 的計算題錯在單位不一致。先把所有單位寫清楚（cm vs m、radian vs degree、g/mol vs kg）

2. **常見陷阱**：
   - Bragg's Law 的 θ 不是 2θ（記得除 2）
   - Scherrer 的 β 要用 **radian**
   - Hooke's Law 的 c 要用 **cm/s** 不是 m/s
   - Reduced mass 的質量要用 kg（不是 g/mol 直接代）

3. **數量級檢查**：
   - IR 波數 100 – 4000 cm⁻¹
   - UV-Vis 波長 200 – 800 nm
   - X-ray 波長 0.5 – 2.5 Å
   - 算出 IR 振動頻率 10 cm⁻¹ 或 100000 cm⁻¹ 一定錯

4. **計算機預設**：
   - 確認是 DEG 模式還是 RAD 模式
   - Scherrer 記得切換成 RAD 或乘 π/180

5. **寫公式再代數字**：先把公式寫清楚，再代入數值，避免算到一半忘記在算什麼

## 最常考的 5 題型（優先順序）

1. **Beer-Lambert Law**（A = εbc，求其中一個變數）— 必考
2. **Bragg's Law**（求 d 或 2θ）— XRD 講義重點
3. **Hooke's Law**（求 C-H/C-D 比值、估振動頻率）— 與 IR 選擇定則結合
4. **波長/波數/能量互換**（λ ↔ v̄ ↔ E）— 光譜基礎
5. **Scherrer Equation**（求晶粒大小）— XRD 考點

**加分題可能出**：
- Tauc plot 求 Eg
- ATR penetration depth
- Ensemble averaging SNR
