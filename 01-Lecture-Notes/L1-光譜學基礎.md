# L1 光譜學基礎

## 核心概念

### 什麼是 Spectroscopy

**Spectroscopy（光譜學）** 研究物質與**電磁輻射（electromagnetic radiation）** 的交互作用；透過分析物質吸收、發射或散射光的行為（隨波長變化），來判定化學組成、物理結構與電子性質。

基本原理：每個原子或分子都有**量子化（quantized）** 的離散能階。光與分子交互作用時會出現三種過程：

- **Absorption（吸收）**：分子吸收能量，電子躍遷到較高能階，或鍵結振動
- **Emission（發射）**：被激發的分子回到較低能階時以光的形式釋放能量
- **Scattering（散射）**：光被分子彈開，有時頻率會改變（例：Raman scattering），取決於分子振動

### 分子總能量分解（關鍵統一公式）

分子的總能量可以分解成三部分：

```
E_total = E_electronic + E_vibrational + E_rotational
```

**三種能階的大小順序**（能量差）：
- **E_electronic**（電子能階）：**最大**，對應 UV-Vis / X-ray 範圍（eV 級）
- **E_vibrational**（振動能階）：**中等**，對應 IR 範圍（0.01–1 eV）
- **E_rotational**（旋轉能階）：**最小**，對應 microwave 範圍（< 0.01 eV）

**為什麼 UV-Vis 譜峰是寬帶？**
因為在每個電子能階上都疊了許多振動子能階，再疊旋轉子能階 → 實際看到的是**很多躍遷的包絡**。

**這條公式就是光譜學的基本地圖**：決定不同波長區段對應什麼樣的激發。

### 光譜儀的三大基本組件

所有光譜儀都由三個方塊組成：

```
Light  →  Sample / Optics  →  Detector
```

- **Light（光源）**：IR / Visible / UV / X-ray
- **Sample / Optics**：發生 absorption、transmission、emission、scattering
- **Detector**：單通道（SCD）或多通道（MCD）

### 光的本質與電磁波譜

光同時具有**波動性**與**粒子性（photons / quanta）**。

**電磁波譜（EM spectrum）涵蓋範圍**（由高能到低能）：
Gamma rays → X-rays → UV → Visible → IR → Microwaves → Radio waves

**可見光範圍**：約 380 nm（紫）~ 780 nm（紅）

**光速定律**：c = λν ≈ 3.00 × 10⁸ m/s（真空中）

- 進入密度較高介質（如玻璃），速度下降 → 波長變短；回到空氣後波長恢復，**頻率 ν 始終不變**

**波數（wavenumber）** v̄：

- 定義：v̄ = 1/λ，單位 cm⁻¹
- 與光子能量、頻率成正比（不像 λ 本身是倒數關係）

**光子能量**：

- E = hν = hc/λ = hc·v̄
- h = 6.63 × 10⁻³⁴ J·s（Planck 常數）
- 1 eV = 1.60 × 10⁻¹⁹ J

**各光譜區段波長範圍**（重要對照）：
| Region | Wavelength |
|---|---|
| UV | 180–380 nm |
| Visible | 380–780 nm |
| Near-IR | 0.78–2.5 μm |
| Mid-IR | 2.5–50 μm |

### 線性 vs 非線性光學（Linear vs Nonlinear Optics）

這是考試**技術類**考點，重點在原理差異。

**Linear optics（線性）**：介質的極化 **P** 隨電場 **E** 線性響應

- 公式：P = ε₀·χ⁽¹⁾·E
- χ⁽¹⁾ 為一階（線性）susceptibility
- 特徵：
  1. 光波之間不交互作用
  2. 光強度弱時
  3. 現象：reflection、refraction、transmission、interference、diffraction、linear absorption、scattering

**Nonlinear optics（非線性）**：光場強度很強時，極化出現高階項

- 公式：P = ε₀·(χ⁽¹⁾E + χ⁽²⁾E² + χ⁽³⁾E³ + …)
- χ⁽²⁾、χ⁽³⁾ 分別為二階、三階 susceptibility
- 特徵：
  1. 光與物質交互作用強度相依，**會產生新頻率**
  2. 高強度光源才會出現（例：lasers）
- 典型現象：
  - **SHG（Second Harmonic Generation，二倍頻）**：ω + ω → 2ω
  - **SFG（Sum Frequency Generation，和頻產生）**：ω_vis + ω_IR → ω_SFG

### 太陽輻射光譜（Solar Radiation Spectrum）

太陽發射譜近似於 **5800 K 的 blackbody radiator**。能量分布：

- UV（<400 nm）：約 7–9%
- Visible（400–700 nm）：約 42–44%（藍綠主導，所以天空是藍色）
- IR（>700 nm）：約 48–53%（主要是熱）

**大氣效應**：

- **Absorption**：特定氣體吸收特定波長。O₃ 吸收大部分 UV，H₂O 與 CO₂ 在 IR 區形成吸收帶
- **Scattering**：Rayleigh scattering（由小分子造成），偏好散射較短波長（藍光）→ 天空藍

### Rayleigh vs Mie Scattering（散射的兩大類型）

散射依**粒子大小與光波長的比例**分兩類：

| 類型 | 粒子大小 | 散射強度對 λ 的依賴 | 散射方向 | 範例 |
|---|---|---|---|---|
| **Rayleigh** | **粒子 ≪ λ**（< λ/10） | **∝ 1/λ⁴**（短波長強） | 幾乎各向同性 | 大氣分子散射太陽光 → 天空藍 / 日落紅 |
| **Mie** | **粒子 ≈ λ 或更大** | 幾乎獨立於 λ | 前向散射為主 | 雲、霧、灰塵、牛奶 |

**重點推論**：
- **為什麼天空是藍的？** → 大氣分子 ≪ 可見光波長 → Rayleigh ∝ 1/λ⁴ → 藍光散射比紅光強 ≈ 10 倍
- **為什麼日落是紅的？** → 陽光斜穿長路徑大氣 → 藍光先被散射掉 → 剩紅光到眼睛
- **為什麼雲是白的？** → 水滴大小 ≈ 可見光波長 → Mie scattering 不選頻率 → 所有顏色均勻散射 → 白
- **為什麼牛奶是白的？** → 脂肪微粒跟可見光同尺度 → Mie 散射

**光譜量測意涵**：
- 混濁 / 懸浮樣品會散射光 → 透射量測虛高吸收
- **UV 區（短波長）對 Rayleigh 更敏感** → 比 visible 區容易被散射干擾
- 需要用 **integrating sphere** 把散射光收回來才能測到真實吸收
- 日地距離處的總輻照度 ≈ **1360 W/m²**

## Absorption 方法與 Beer-Lambert Law

### Absorption 過程

入射輻射功率 P₀ 經過樣品後變為透射功率 P（P < P₀）。差值被 analyte 吸收，**吸收能量必須精確對應於兩能階間差值**。

關鍵定義：

- **Transmittance**：T = P/P₀
- **Absorbance**：A = −log T = log(P₀/P)

### Beer-Lambert Law（Beer's Law）

**A = εbc**（無單位量 → 濃度 × 路徑長 × molar absorptivity）

| 符號 | 意義                | 單位         |
| ---- | ------------------- | ------------ |
| A    | absorbance          | 無單位       |
| ε    | molar absorptivity  | L·mol⁻¹·cm⁻¹ |
| b    | 光程（path length） | cm           |
| c    | 濃度                | mol/L        |

### Beer's Law 的限制（考點！）

1. **Real limitations（真實限制）**
   - Beer's Law 只對**稀溶液**成立，因此本質上是**極限律（limiting law）**
   - 濃度 > 0.01 M 時，吸收分子彼此太近，會影響電荷分布與其他吸收分子的 ε，造成偏離線性
   - 高電解質濃度時也會有類似效應

2. **Chemical deviations（化學偏差）**
   - 溶質在溶液中發生解離（dissociation）、締合、與溶劑反應 → 改變吸收物種

3. **Instrumental deviations（儀器偏差）**
   - **Polychromatic radiation（多色光）**：若使用非單色光，每個波長有不同 ε
   - **Stray light（雜散光）**

### Stray Light（雜散光）

- 定義：來自儀器、**位於所選波長頻帶外**的輻射，通常來自光柵、鏡面、濾鏡、窗片的散射與反射
- 公式：A' = log[(P₀ + P_s)/(P + P_s)]
- 效應：雜散光使**表觀吸收度低於真實值**
- 現代儀器 stray light 可壓到 < 0.5%；大多數儀器 A > 2.0 就難以測量
- 若 stray light 很高，甚至 A > 1.0 時就會偏離 Beer's Law

## 發射光譜（Emission Spectra）

### 三種發射譜型

一張發射光譜圖上通常同時疊加三種：

1. **Line spectrum（線光譜）**：來自激發的**原子**個體輻射
2. **Band spectrum（帶光譜）**：來自小分子或自由基
3. **Continuum spectrum（連續光譜）**：來自熱的固體（blackbody）

### 連續光譜與 Blackbody radiation

- 熱固體（如 carbon、tungsten）被加熱到白熾狀態時產生
- 溫度越高 → 最大發射波長越短（Wien's law 的定性描述）
- 2000 K、3000 K、4000 K、6000 K 的曲線：峰值往短波長移
- 常見連續光源：Xenon arc、Carbon arc、Tungsten lamp、Nernst glower

## 波長選擇器（Wavelength Selector）

光譜儀中用來**從寬光源中分離出特定波長或窄波段**的關鍵元件。

### 核心功能

1. **Selectivity（選擇性）**：隔離出 analyte 獨有的吸收/發射波段
2. **Sensitivity（靈敏度）**：阻擋背景雜散輻射，提升 SNR
3. **Resolution（解析度）**：由 **effective bandwidth** 決定（透射率為最大值 50% 時的波長範圍寬度）

### 兩大類型

#### (1) Filters（濾鏡）

最簡單的固定波長選擇器。

**Absorption Filters（吸收式濾鏡）**

- 機制：由有色玻璃、塑膠、或明膠染料組成，選擇性吸收不要的波長並轉成熱
- 優點：便宜、簡單
- 缺點：頻寬較寬、透射效率較低

**Interference (Dichroic) Filters（干涉式濾鏡）**

- 機制：使用薄膜塗層，透過**建設性/破壞性干涉**讓想要的波長通過，其他被反射
- 原理：**Snell's Law** n₁ sin θ₁ = n₂ sin θ₂ + thin-film interference
- 優點：bandwidth 非常窄（可達 1–10 nm），透射率高
- 缺點：貴；**對入射角度敏感**

**Filter 依頻譜作用分類**：

- **Bandpass**：只讓一段窄頻通過
- **Longpass**：阻擋短波、通過長波（cut-on）
- **Shortpass**：阻擋長波、通過短波（cut-off）
- **Notch**：bandpass 相反，阻擋極窄一段、其他全通

#### (2) Monochromators（單色儀）

可**電子調諧、連續掃瞄**整個波長範圍，比 filter 靈活。

**Prism monochromator**：

```
Entrance slit → Collimating lens → Prism → Focusing lens → Exit slit → Focal plane
```

利用 prism 的色散

**Grating monochromator**：

- 使用 **reflection grating（反射光柵）** + concave mirrors
- 利用 **diffraction（繞射）** 把不同波長分開（echellette-type grating）
- **高密度光柵 → 高色散 → 高解析度**

### Filter vs Monochromator 比較表（重要！）

| 特徵   | Filter-Based                | Monochromator-Based       |
| ------ | --------------------------- | ------------------------- |
| 彈性   | 低；換波長要換硬體          | 高；電子可調諧            |
| 靈敏度 | 較**高**（透光量多）        | 較**低**（狹縫/鏡面損耗） |
| 成本   | 便宜簡單                    | 貴且複雜                  |
| 應用   | 例行分析、field instruments | 研究、全光譜掃瞄          |

## 偵測器（Detectors）

**Detector** = 把光信號（光）轉成可量測的電信號（電流/電壓），是儀器的「眼睛」。

### CCD（Charge-Coupled Device）

- 高靈敏半導體感測器，把 photons 轉成 electrons
- 科學研究的**gold standard**，靈敏度高、noise 低

### CCD Array

- 多通道偵測器，由數千個獨立感光像素排成 1D 或 2D
- 優於傳統**單通道掃瞄偵測器**：
  - 傳統 scanning detector → 一次一個波長
  - CCD array → **同時擷取整張光譜**
- 搭配 grating：grating 把光散成不同波長 → 投影到 CCD → 每個像素對應一個波長

## Signal-to-Noise Ratio（SNR）

**SNR 定義**：真實化學信號相對於背景電子/光學雜訊的比值。

### SNR 提升法：Ensemble Averaging

公式：(S/N)\_n = √n · (S/N)\_i

- **n** = 資料點數（取樣次數）
- SNR 與**資料點數的平方根**成正比
- 相同原理也適用於 boxcar averaging、digital filtering

### 儀器雜訊來源（四類）

1. **Thermal / Johnson noise（熱雜訊）**：可以透過
   - 降低電路電阻
   - 降低元件溫度來降（例：把 UV-Vis 光電二極體從室溫 298 K 冷到液氮 77 K 可大幅減少）
2. **Shot noise（散粒雜訊）**
3. **Flicker / 1/f noise**
4. **Environmental noise**

## IR Absorption Spectroscopy（初步介紹，細節在 L2/IR 講義）

### 原理

**IR Absorption Spectroscopy**：透過測量分子對特定 IR 頻率的吸收，辨識化學物質或官能基。原理是分子吸收符合其**內部振動能量**的光。

### IR 區段

| Region        | 波長          | 波數 cm⁻¹    | 頻率 Hz           |
| ------------- | ------------- | ------------ | ----------------- |
| Near          | 0.78–2.5 μm   | 12800–4000   | 3.8×10¹⁴–1.2×10¹⁴ |
| Middle        | 2.5–50 μm     | 4000–200     | 1.2×10¹⁴–6.0×10¹² |
| Far           | 50–1000 μm    | 200–10       | 6.0×10¹²–3.0×10¹¹ |
| **Most used** | **2.5–15 μm** | **4000–670** | —                 |

### Selection Rule for IR（關鍵考點！）

**分子振動時必須造成 dipole moment（偶極矩）的改變才會 IR active。**

- **Permanent dipole 有 → IR active**：例 OH 鍵
- **無 permanent dipole → IR inactive**：例 O₂、N₂、Cl₂

### 分子振動模式（Vibration modes）

**Stretching（伸縮）**：

- Symmetric stretch
- Asymmetric stretch

**Bending（彎曲）**：

- In-plane rocking
- In-plane scissoring
- Out-of-plane wagging
- Out-of-plane twisting

### 振動自由度

- 分子有 N 個原子 → 共 3N 自由度
- 扣掉 3 個平移 + 3 個轉動（線型分子只扣 2 個轉動）
  - 非線型分子：**3N − 6** 個振動模式
  - 線型分子：**3N − 5** 個振動模式

### CO₂ 範例（考點）

- 線型三原子分子，3×3 − 5 = **4 個 normal modes**
- (a) 對稱伸縮：兩個 C=O 偶極反向 → 淨偶極不變 → **IR inactive**
- (b) 反對稱伸縮：淨偶極改變 → **IR active**
- (c), (d) 兩個彎曲模式**degenerate（簡併）**（頻率相同）→ IR active
- 所以 CO₂ 的 IR 光譜**只有兩條吸收帶**

### H₂O 範例（考點）

- 非線型三原子分子，3×3 − 6 = **3 個 normal modes**
- 角度 104.5°，兩個 O–H 偶極不對消 → 分子有**永久淨偶極**
- 三個模式都 IR active：
  - Symmetric stretch
  - Asymmetric stretch
  - Scissoring bend
- IR 光譜**有三條吸收帶**
